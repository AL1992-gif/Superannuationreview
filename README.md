# SuperScope Australia

A mobile-friendly static web app for comparing indicative Australian superannuation fees, historical performance inputs and long-term balance projections.

## Features

- Compare major public-offer fund options
- Recalculate estimated annual fees for different balances
- Filter by active, indexed and lifecycle strategies
- Compare 1-year, 5-year and 10-year performance inputs
- Shortlist up to three options
- Project future and inflation-adjusted balances
- Edit the starter dataset and save changes locally in the browser
- No backend, database or paid API required

## Data warning

The bundled dataset is an editable research snapshot, not a live feed or financial recommendation. Disclosure dates, risk levels and asset allocations may differ. Check current PDS documents and official tools before making a decision:

- ATO YourSuper comparison tool
- APRA MySuper Product Heatmap
- ASIC Moneysmart guidance

## Deployment

The included GitHub Actions workflow deploys the repository to GitHub Pages after each push to `main`.
